Content - MGet
=============

This plugin will output the current user information or guest or 0 for user id if not logged in. Works in content only.

Compatiable with Joomla 1.5/1.6

Usage
_____

Input			Outputs
{mgetuid}		User ID
{mgetuser}		User Name
{mgetuname}		Users Name
{mgetueml}		Users email
